Instruction

  1. Click Add a question button
  2. Click Choose File button
  3. Once decide which file to upload, click Upload button
  4. Then, at Enter a question section, enter your question
    • If the question has answers, click Add to add an answer. Once decide, click Submit
    • If this question does not have answers, skip Add an answer for multiple choices
  5. After setting up a question, click Submit Question
  6. Your input will be shown at the bottom of the page
  7. If decide to delete a question, click DELETE button
  8. If decide to insert a question at Insert Question Here section, click Insert Question Here button
  9. Repeat the above steps for every question
  10. Once set up all the questions for an experiment, click Submit Experiment button
